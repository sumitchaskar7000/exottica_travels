# TravelWebsitePro Frontend

Next.js 14 (App Router) + Tailwind CSS frontend for the TravelWebsitePro travel agency website.

## Getting started

```bash
cd frontend
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

You can configure the backend API base URL via:

```bash
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```
